# LPDP CTRL+F

**Control your Future** - Cari universitas impianmu untuk beasiswa LPDP.

From awardee to calon awardee. Pro-bono dan tidak menerima mentorship berbayar.

## Features

- 🔍 **Pencarian Universitas** - Cari dari 28.000+ program studi LPDP
- 📊 **Filter STEM/SHARE** - Filter berdasarkan jenis beasiswa
- 📄 **Resume Analyzer** - Analisis CV dengan AI untuk rekomendasi universitas
- 📥 **Download Panduan** - Unduh panduan resmi LPDP STEM & SHARE

## Tech Stack

- **Frontend**: Vanilla HTML/CSS/JS
- **Backend**: Python Flask
- **AI**: OpenAI GPT-4

## Quick Start

```bash
# 1. Setup environment
cp .env.example .env
# Edit .env dengan OPENAI_API_KEY

# 2. Install dependencies
pip install -r backend/requirements.txt

# 3. Start servers
./start.sh
```

Website: http://localhost:8000
API: http://localhost:8001
